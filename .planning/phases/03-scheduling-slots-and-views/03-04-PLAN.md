**EditLessonSlot (SCH-02):**
- `EditLessonSlotCommand` implements `ICommand<Unit>`: properties `Id` (Guid), `TeacherId` (Guid), `StartTime` (DateTimeOffset), `EndTime` (DateTimeOffset)
- `EditLessonSlotCommandHandler`: inject `ILessonSlotRepository`, `IUnitOfWork`
  1. Find slot: `var slot = await slotRepo.FindByIdAsync(command.Id, ct)` — throw `NotFoundException` if null
  2. Check conflict with self-exclusion (per D-06): `var hasConflict = await slotRepo.HasConflictAsync(command.TeacherId, command.StartTime, command.EndTime, command.Id, ct)` — pass `command.Id` as excludedSlotId
  3. If `hasConflict`, throw `ValidationException` with message: `"Teacher is already booked at the requested time"`
  4. Apply changes: if teacher changed call `slot.ChangeTeacher(command.TeacherId)`, if time changed call `slot.Reschedule(command.StartTime, command.EndTime)`
  5. `await unitOfWork.SaveEntitiesAsync(ct);`
- `EditLessonSlotCommandValidator`: `Id` not default, `TeacherId` not default, `StartTime` before `EndTime`
- `EditLessonSlotEndpoint`: PUT `/lesson-slots/{id}`, `.RequireAuthorization(SchedulingPermissions.EditLessonSlot)`
