# Main App - Edvantix

Основное приложение (панель управления) для Edvantix - системы управления онлайн школой.

## Запуск

```bash
bun dev
```

Приложение будет доступно по адресу: http://localhost:3002

## Технологии

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

## Структура

- `/app` - страницы приложения (App Router)
- `/components/ui` - UI компоненты shadcn/ui
- `/lib` - утилиты и хелперы
