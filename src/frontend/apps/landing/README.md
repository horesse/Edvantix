# Landing - Edvantix

Landing page с SSR для Edvantix - системы управления онлайн школой.

## Запуск

```bash
bun dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Технологии

- Next.js 15 (App Router с SSR)
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

## Особенности

- Server-Side Rendering (SSR) по умолчанию
- SEO оптимизация
- Оптимизированные метаданные

## Структура

- `/app` - страницы приложения (App Router)
- `/components/ui` - UI компоненты shadcn/ui
- `/lib` - утилиты и хелперы
