**CreateGroup:**
- `CreateGroupCommand` implements `ICommand<Guid>`: properties `Name` (string), `MaxCapacity` (int), `Color` (string)
- `CreateGroupCommandHandler`: inject `IGroupRepository`, `ITenantContext`, `IUnitOfWork`. Create `new Group(command.Name, tenantContext.SchoolId, command.MaxCapacity, command.Color)`, add to repo, `await unitOfWork.SaveEntitiesAsync(ct)`, return `group.Id`
- `CreateGroupCommandValidator`: `Name` not empty, max length 200; `MaxCapacity` greater than 0; `Color` not empty, max length 50
- `CreateGroupEndpoint`: POST `/groups`, `Produces<Guid>(201)`, `.WithName("CreateGroup").WithTags("Groups").MapToApiVersion(ApiVersions.V1).RequireAuthorization(SchedulingPermissions.CreateGroup)`

**UpdateGroup:**
- `UpdateGroupCommand` implements `ICommand<Unit>`: properties `Id` (Guid), `Name` (string), `MaxCapacity` (int), `Color` (string)
- `UpdateGroupCommandHandler`: inject `IGroupRepository`, `IUnitOfWork`. Find group by Id, throw `NotFoundException` if null, call `group.Update(command.Name, command.MaxCapacity, command.Color)`, save
- `UpdateGroupCommandValidator`: `Id` not default, `Name` not empty max 200, `MaxCapacity` > 0, `Color` not empty max 50
- `UpdateGroupEndpoint`: PUT `/groups/{id}`, `.RequireAuthorization(SchedulingPermissions.UpdateGroup)`

**DeleteGroup:**
- `DeleteGroupCommand` implements `ICommand<Unit>`: property `Id` (Guid)
- `DeleteGroupCommandHandler`: find group by Id, throw `NotFoundException` if null, call `group.Delete()` (soft delete per D-02), save
- `DeleteGroupEndpoint`: DELETE `/groups/{id}`, `.RequireAuthorization(SchedulingPermissions.DeleteGroup)`
