  1. Validate group exists: `var group = await groupRepo.FindByIdAsync(command.GroupId, ct)` â€” throw `NotFoundException` if null

  3. If `hasConflict` is true, throw a `ValidationException` with message: `"Teacher is already booked at the requested time"`
  4. Create: `var slot = new LessonSlot(tenantContext.SchoolId, command.GroupId, command.TeacherId, command.StartTime, command.EndTime)`
  5. `slotRepo.Add(slot); await unitOfWork.SaveEntitiesAsync(ct); return slot.Id;`
- `CreateLessonSlotCommandValidator`: `GroupId` not default, `TeacherId` not default, `StartTime` must be before `EndTime`, both must not be default

- `EditLessonSlotCommand` implements `ICommand<Unit>`: properties `Id` (Guid), `TeacherId` (Guid), `StartTime` (DateTimeOffset), `EndTime` (DateTimeOffset)
- `EditLessonSlotCommandHandler`: inject `ILessonSlotRepository`, `IUnitOfWork`
  1. Find slot: `var slot = await slotRepo.FindByIdAsync(command.Id, ct)` â€” throw `NotFoundException` if null
  2. Check conflict with self-exclusion (per D-06): `var hasConflict = await slotRepo.HasConflictAsync(command.TeacherId, command.StartTime, command.EndTime, command.Id, ct)` â€” pass `command.Id` as excludedSlotId

  4. Apply changes: if teacher changed call `slot.ChangeTeacher(command.TeacherId)`, if time changed call `slot.Reschedule(command.StartTime, command.EndTime)`
  5. `await unitOfWork.SaveEntitiesAsync(ct);`

- `EditLessonSlotEndpoint`: PUT `/lesson-slots/{id}`, `.RequireAuthorization(SchedulingPermissions.EditLessonSlot)`
