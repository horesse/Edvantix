**GetScheduleQueryHandler (per D-07, D-08, D-09):**

Inject: `SchedulingDbContext` (direct context access for complex query), `ITenantContext`, `ClaimsPrincipal` (for profileId), and the Organizations gRPC client (`PermissionsGrpcService.PermissionsGrpcServiceClient` from Chassis proto).

The handler implements the following logic:

1. Extract caller's `profileId` from `ClaimsPrincipal` using the same `TryGetProfileId()` pattern used in Organizations.

2. The endpoint already has `.RequireAuthorization(SchedulingPermissions.ViewSchedule)` as baseline gate (one gRPC call via PermissionRequirementHandler). Inside the handler, make ONE additional gRPC call to determine if the caller is a manager:
```csharp
var isManager = (await grpcClient.CheckPermissionAsync(new CheckPermissionRequest
{
    UserId = profileId.ToString(),
    SchoolId = tenantContext.SchoolId.ToString(),
    Permission = SchedulingPermissions.CreateLessonSlot  // proxy for manager level
})).HasPermission;
```
This second call is served from HybridCache so it is cheap.

3. Build the base query — join `LessonSlots` with `Groups` on `GroupId` to get group name and color, filter by date range:
```csharp
var query = from slot in dbContext.LessonSlots
            join group in dbContext.Groups on slot.GroupId equals group.Id
            where slot.StartTime >= command.DateFrom && slot.StartTime < command.DateTo
            select new { slot, group };
```

4. Apply data filtering based on permission level:
   - **Manager** (`isManager == true`): No additional filter — return ALL tenant's slots (tenant filter already applied by HasQueryFilter). Include `TeacherId`, `TeacherName` (placeholder), `StudentCount`.
   - **Teacher** (not manager, but caller's profileId matches TeacherId): Filter to `slot.TeacherId == profileId`. Include `StudentCount`. Set `TeacherId = null`, `TeacherName = null` (teacher already knows who they are).
   - **Student** (neither manager nor teacher): Filter to slots where `groupId IN (SELECT groupId FROM GroupMemberships WHERE StudentId == profileId)`. Include `TeacherId`, `TeacherName`. Set `StudentCount = null`.

   To determine if the caller is a teacher: check if any slot in the result set has `TeacherId == profileId`. If no slots match, fall through to student.

   Implementation approach:
   ```csharp
   if (isManager)
   {
       // Return all slots with manager-level fields
       return await query.Select(x => new ScheduleSlotDto(
           x.slot.Id, x.slot.StartTime, x.slot.EndTime,
           x.slot.GroupId, x.group.Name, x.group.Color,
           x.slot.TeacherId, x.slot.TeacherId.ToString(), // placeholder name
           dbContext.GroupMemberships.Count(m => m.GroupId == x.slot.GroupId)
       )).ToListAsync(ct);
   }

   // Check if caller is a teacher for any slot in range
   var isTeacher = await dbContext.LessonSlots
       .AnyAsync(s => s.TeacherId == profileId
           && s.StartTime >= command.DateFrom && s.StartTime < command.DateTo, ct);

   if (isTeacher)
   {
       return await query
           .Where(x => x.slot.TeacherId == profileId)
           .Select(x => new ScheduleSlotDto(
               x.slot.Id, x.slot.StartTime, x.slot.EndTime,
               x.slot.GroupId, x.group.Name, x.group.Color,
               null, null, // no teacher info for teacher view
               dbContext.GroupMemberships.Count(m => m.GroupId == x.slot.GroupId)
           )).ToListAsync(ct);
   }

   // Student: filter to groups where student is a member
   var studentGroupIds = dbContext.GroupMemberships
       .Where(m => m.StudentId == profileId)
       .Select(m => m.GroupId);

   return await query
       .Where(x => studentGroupIds.Contains(x.slot.GroupId))
       .Select(x => new ScheduleSlotDto(
           x.slot.Id, x.slot.StartTime, x.slot.EndTime,
           x.slot.GroupId, x.group.Name, x.group.Color,
           x.slot.TeacherId, x.slot.TeacherId.ToString(), // placeholder name
           null // no student count for student view
       )).ToListAsync(ct);
   ```

5. Return `List<ScheduleSlotDto>`.
